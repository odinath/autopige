# Installation
- Make sure the 'npm' command is properly installed on your local environment.
- Access the newly-cloned 'ubitest' folder from a terminal prompt.
- On your initial run, install the webapp with: "> npm install"
- From there on, just launch the webapp with: "> npm start"
- Access the following URL with your web browser: "http://localhost:8080"
